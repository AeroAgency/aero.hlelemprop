# aero.hlelemprop
# Установка
### c помощью composer
`composer require aeroidea/aero.hlelemprop dev-master`
### в ручную
1. Скачиваем архив с модулем
2. распаковываем в `www/local/modules`

# После установки
 
нужно зайти в `Marketplace -> Установленные решения`
