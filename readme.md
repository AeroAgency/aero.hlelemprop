# Документация модуля aero.hlelemprop
## Описание
В "1C-Битрикс" есть тип свойства элемента инфоблока "Справочник". Этот тип свойства сильно нагружает админку, особенно, если в системе есть highload-блоки с колличеством элементов >1000.<br/>
Данный модуль добавляет пользовательский тип свойства - "Привязка к элементу справочника". Привязка работает аналогично стандартному типу свойства, но обращается к БД при необходимости отредактировать значение конкретного свойства. Это делает возможным работу с элементами ИБ в административной части сайта без зависаний и ускоряет загрузку страниц админ-панели.
## Установка
### c помощью composer
`composer require aeroidea/aero.hlelemprop`

### в ручную
1. Скачиваем архив с модулем
2. распаковываем в `www/local/modules`
